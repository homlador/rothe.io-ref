# Tastaturereignisse
---

~~~ python
def on_key_down(taste):
~~~
wird aufgerufen, wenn eine Taste gedrückt wird.

~~~ python
def on_key_up(taste):
~~~
wird aufgerufen, wenn eine Taste losgelassen wird.

Die Variable `taste` kann folgende Werte annehmen:
